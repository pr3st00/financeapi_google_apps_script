DESCRICAO

  • Planilha para controle e rebalanceamento de ativos de renda fixa e variavel, mantendo a alocacao macro desejada.
  • Cotacoes e indicadores sao atualizados automaticamente pelo Google Finance e por API customizada.
  • Os ativos sao ordenados conforme a discrepancia de alocacao, destacando os melhores alvos de aporte.
  • Suporte a quarentena de ativos via checkbox "Q" para excluir temporariamente itens dos calculos.
  • Abas auxiliares: Historico e Resultados para analise de variacoes.

VISAO GERAL

  • Insira os objetivos de alocacao na linha 3 (META) e possiveis fundos.
  • Ao registrar um aporte, a planilha calcula quanto cada categoria deve receber.
  • Cores indicam aumentar, reduzir ou montante OK (tolerancia de 10%).
  • O checkbox Mask esconde todos os montantes das abas.

FIIS

  • Fundos imobiliarios com cadastro manual; copie uma linha existente e atualize ticker, SEG, ADM, VACAN, ATIVOS e nota.
  • Campo ATUAL: cotas * cotacao atual (ex: =42*C3). Use "Atualizar dados" no menu Investimentos para sincronizar indicadores.
  • Colunas importantes: APORTE, # DIAS, TAXA SELIC, IR, P, %, e %G.
  • Use "ORDENAR" para destacar ativos atrasados.

ACOES NAC

  • Acoes e ETFs nacionais com cadastro manual.
  • Colunas: APORTE, # DIAS, CORRETAGEM, % categoria, %G carteira e C (classificacao).
  • CONSENSO e um campo manual (C/M/V); nao delete colunas para evitar que scripts quebrem.
  • Use o botao "ORDENAR" para destacar ativos com maior discrepancia.

ACOES INT

  • Stocks e BDRs; cadastro manual e colunas similares as Acoes Nacionais.
  • Use "Investimentos -> Acoes Internacionais -> Ordenar" para ordenar os ativos.

HISTORICO

  • Historico de cotacoes e medias de notas.
  • Mostra variacoes em 30, 60, 120 e 360 dias e potencial de UPSIDE.
  • Ordene por data de aporte, upside e variacoes.
